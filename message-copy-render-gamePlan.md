# Game Plan: Message Copy Render

## Aufgaben
- Nachrichtenmodell so erweitern, dass volle und gekuerzte Darstellung getrennt sind.
- Sammelkopie auf eine feste Obergrenze begrenzen und immer die neuesten sichtbaren Nachrichten verwenden.
- Pro Nachricht eine eigene Kopieraktion fuer die komplette Nachricht einfuehren.
- Renderer fuer lange Werte auf 150 Zeichen begrenzen, ohne die Kopierdaten zu verlieren.
- Aenderungen mit Build pruefen und Ergebnis dokumentieren.

## Offene Punkte
- Sinnvolle Obergrenze fuer Sammelkopie festlegen.
- Ganze Nachricht beim Einzelkopieren als kompaktes, nachvollziehbares Format zusammensetzen.

## Leitplanken
- Bestehendes Filterverhalten erhalten.
- Anzeige und Kopiertext nicht vermischen.
- Aenderungen klein und auf monitor begrenzt halten.
